# 📊 Outlier-Removal-IQR-ML

## 📌 Description

A machine learning project demonstrating how to detect and remove outliers from a dataset using the IQR (Interquartile Range) method.

---

## 📌 Objective

To improve data quality by identifying and removing extreme values (outliers) that can negatively impact analysis and model performance.

---

## ⚙️ Workflow

* Import required libraries (pandas, matplotlib, seaborn)  
* Load dataset  
* Analyze ApplicantIncome
* Detect outliers using IQR method
* Remove outliers from the dataset 
* Visualize data before and after removal using boxplot
  
---

## 🚀 Features

* Detects outliers using IQR method
* Removes outliers from ApplicantIncome
* Shows data before and after cleaning
* Uses visualization for better understanding
* Beginner-friendly implementation  

---

## 🛠️ Technologies Used

* Python  
* Pandas  
* Matplotlib  
* Seaborn  
* Jupyter Notebook
